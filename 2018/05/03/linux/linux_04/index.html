（ssh：连接到主机本地主机端口22：连接被拒绝）

2.安装ssh-server

3.启动服务

启动成功会提示：

[ ok ] Starting ssh (via systemctl): ssh.service.

4.检查服务是否启动成功

若有下面提示则启动成功
6455 ? 00:00:00 sshd
